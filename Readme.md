使用说明:

System.Net实现浏览网页后获取源码,使用字符串对源码处理并提取成虾米音乐可使用的格式

在主界面的文本框中输入thbwiki的专辑界面即可完成信息提取,可在生成的专辑信息界面直接对信息进行修改~
